## Requirements

- PHP 8.1+

## Installation

- Add `php-lsp/rpc-protocol-jsonrpc` as composer dependency.

```json
{
    "require": {
        "php-lsp/rpc-protocol-jsonrpc": "^1.0"
    }
}
```
